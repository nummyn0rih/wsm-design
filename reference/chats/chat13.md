ВКЛАДКА «Основные данные»:
- Название, ИНН, юр.форма, юр.адрес, фактический адрес
- Статус: активный / архив (переключатель)
- Теги (мульти-выбор: «приоритетный», «новый», «проблемный»)
- Дата начала сотрудничества
- Заметка (textarea)
- Кнопка «Скорректировать остаток тары» (плейсхолдер действия)

ВКЛАДКА «Контакты»:
- Список контактов карточками: ФИО, должность, телефон (tel:),
  email (mailto:), пометка «основной»
- Кнопка «+ Добавить контакт»
- Общие заметки по коммуникации (textarea)
- Показать минимум 3 контакта

ВКЛАДКА «Контракты»:
- Список всех контрактов поставщика, группировка по сезонам
  (сворачиваемые секции)
- Каждый контракт — строка-карточка: сезон, сырьё (пилюли),
  объём, прогресс-бар выполнения, статус
- Текущий сезон раскрыт, прошлые — свёрнуты
- Кнопка «+ Создать контракт для этого поставщика»
- Клик по карточке → переход на страницу контракта (плейсхолдер)
- Показать текущий сезон с 2–3 контрактами + 1 свёрнутый прошлый сезон

ВКЛАДКА «История отгрузок»:
- Фильтры сверху: период (от/до), сырьё (мульти), статус качества
- Сортируемая таблица: дата, сырьё (пилюля), машина/водитель, вес,
  статус качества (иконка как в задаче 8.1в)
- Подитог снизу
- Кнопка «Экспорт» (Excel) и «🖨 Печать»
- Показать минимум 10 строк, разные виды сырья и статусы качества

ВКЛАДКА «Тара»:
- Сводка-плашки сверху: «На заводе: X · У поставщика: Y · Лом: Z»
- Журнал движений тары по этому поставщику — таблица как на
  странице «Тара · Журнал движений», но отфильтрованная
- Кнопка «+ Зафиксировать движение тары»
- Показать 6–8 движений

ВКЛАДКА «Качество»:
- KPI-плашки сверху: средний % брака за сезон, средний % нестандарта,
  средний % к оплате
- График «% брака по месяцам» (line chart, апрель–октябрь)
- Таблица «Разбивка по сырью»: сырьё → средний % брака /
  нестандарта / к оплате
- Если у поставщика есть огурцы — мини-виджет «Распределение по
  калибрам» (стек-бар по месяцам)
- Показать поставщика с огурцами и томатами

Шапка карточки поставщика (название, ИНН, статус, KPI-блок)
остаётся прежней. Меняется только содержимое вкладок.
