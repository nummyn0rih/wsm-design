ЗАДАЧА: добавить новую модалку «Качество приёмки» для учёта данных
качества по позиции отгрузки.

Модалка вызывается по иконке 📊 рядом с № акта в карточке позиции
отгрузки. В этой задаче — только сама модалка, два варианта рядом
на демо-экране.

Структура модалки:

1. Блок «Базовый расчёт» (всегда):
   - Вес факт, кг — input
   - Брак, кг — input, рядом авто-% от факта
   - Нестандарт, кг — input, рядом авто-% от факта
     · Чекбокс «Оплачивается отдельно» → разворачивает поле
       «Цена нестандарта, ₽/кг»
   - Вес к оплате, кг (авто, read-only):
     = факт − брак − нестандарт (если нестандарт НЕ оплачивается)
     = факт − брак (если нестандарт оплачивается)
   - % к оплате (авто) от факта

2. Блок «Калибры» (только если у сырья есть параметры качества):
   - Редактируемые поля для параметров с ролью
     «Детализирует вес к оплате» — каждое с весом в кг и авто-%
   - Сумма редактируемых полей должна равняться весу к оплате —
     валидация со строкой статуса:
     · «Σ: 4 150 / 4 150 ✓» — зелёная
     · «Σ: 4 000 / 4 150 ✗ расхождение 150 кг» — красная
   - Read-only поле для параметра с ролью «Дублирует поле Нестандарт» —
     значение зеркалит поле «Нестандарт» из базового блока,
     помечено иконкой 🔗 с тултипом «= Нестандарт»

3. Блок «Документ»:
   - № акта (input)
   - Загрузка PDF акта (зона drag&drop + кнопка)
   - Комментарий (textarea)

4. Кнопки внизу: «Отмена», «Сохранить».
