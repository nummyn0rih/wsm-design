Система учёта отгрузок овощного сырья от поставщиков на переработку.
Текущее состояние — в файле Wireframes.html в проекте.

ВИЗУАЛЬНЫЙ ЯЗЫК (сохранять во всех новых экранах):

Палитра сырья: огурцы — зелёный, черри — красно-розовый,
томаты — оранжевый, патиссоны — жёлтый, халапеньо — тёмно-зелёный,
перец — красно-зелёный градиент, баклажан — фиолетовый.

Чипы статуса: «Запланировано» — жёлтый + полосатая штриховка строки,
«Отправлено» — зелёный.

Пилюли сырья, оверлейный скроллбар, типографика — как в существующих
экранах.

Tailwind через CDN, иконки lucide или эмодзи.

РОЛИ:

Пользователь: главная (просмотр, поиск, фильтры, колонки,
toggle Таблица/Heatmap, Excel, печать), справочник «Водители»
(просмотр + поиск + фильтр по ТК), модалка карточки водителя,
мобильный просмотр.

Админ: всё вышеперечисленное + создание/редактирование везде,
тара, ингредиенты, контракты, аналитика, остальные справочники.

Структура страницы:

1. Заголовок «Уведомления» + счётчик непрочитанных.

2. Тулбар:
   - Две вкладки: «Требуют действия (N)» / «Информационные (N)»
   - Фильтр-чипы по типу: все / 📦 тара / 🚛 машины / 📊 качество /
     📑 контракты / ⚠ алерты
   - Переключатель «Все / только непрочитанные»
   - Поиск по тексту
   - Действия справа: «Отметить все прочитанными»,
     «Архивировать прочитанные»

3. Список уведомлений (карточки в одну колонку):
   - Иконка типа слева
   - Заголовок + описание
   - Дата/время
   - Ссылка на связанный объект (поставщик/контракт/отгрузка)
   - Кнопка действия (для «Требуют действия»): «Подтвердить»,
     «Привязать машину», «Открыть», «Внести данные» и т.п.
   - Точка слева для непрочитанных
   - Чекбокс для массовых действий

Примеры уведомлений во вкладке «Требуют действия» (минимум 5):
- Подтвердить доставку тары поставщику «Генералов» (отправлено
  3 дня назад)
- Привязать машину к запланированной отгрузке на завтра (Цой, огурцы)
- Согласовать акт переработки № 148
- Внести данные качества для отгрузки № 412 (приехала 2 дня назад)
- Контракт с «Иванов» приближается к выполнению (92%)

Примеры во вкладке «Информационные» (минимум 4):
- Контракт с «Петров» выполнен на 90%
- Нет поставок от «Сидоров» более 7 дней
- Перевыполнение контракта «Кузнецов» на 15%
- Высокий % брака от «Морозов» (12% за неделю)

Мобильная адаптация: те же вкладки, карточки в одну колонку,
тулбар сворачивается в кнопку фильтров.

Страницу добавить в конец файла, в новый раздел «Уведомления».
