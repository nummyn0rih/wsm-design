ВИЗУАЛЬНЫЙ ЯЗЫК (сохранять во всех новых экранах):



Палитра сырья: огурцы — зелёный, черри — красно-розовый,
томаты — оранжевый, патиссоны — жёлтый, халапеньо — тёмно-зелёный,
перец — красно-зелёный градиент, баклажан — фиолетовый.

Чипы статуса: «Запланировано» — жёлтый + полосатая штриховка строки,
«Отправлено» — зелёный.

Пилюли сырья, оверлейный скроллбар, типографика — как в существующих
экранах.

Tailwind через CDN, иконки lucide или эмодзи.


РОЛИ:



Пользователь: главная (просмотр, поиск, фильтры, колонки,
toggle Таблица/Heatmap, Excel, печать), справочник «Водители»
(просмотр + поиск + фильтр по ТК), модалка карточки водителя,
мобильный просмотр.

Админ: всё вышеперечисленное + создание/редактирование везде,
тара, ингредиенты, контракты, аналитика, остальные справочники.


ПРАВИЛА РАБОТЫ:



Не переписывай существующие экраны, если в задаче явно не сказано
обратное.

Новые экраны добавляй в конец файла, в логических разделах.

Никакой реальной логики — статичные wireframes с моковыми данными.

ЗАДАЧА: расширить существующий справочник «Сырьё» — добавить
в карточку вида сырья блок «Параметры качества».

В карточке вида сырья (под основными полями) добавить блок
«Параметры качества»:

1. Таблица параметров со столбцами:
   - Название (текст)
   - Единица (по умолчанию «кг»)
   - Роль — выпадающий список:
     · «Детализирует вес к оплате» — сумма таких параметров
        должна сходиться с весом к оплате
     · «Дублирует поле Нестандарт» — зеркалит базовое поле
        «Нестандарт» (может быть только один параметр с этой ролью)
     · «Дублирует поле Брак» — на будущее
     · «Информационный» — не участвует в расчётах
   - Кнопка удаления параметра

2. Кнопка «+ Добавить параметр».

3. Подсказка под блоком: «Параметры качества определяют структуру
   модалки приёмки для этого сырья. Если параметров нет — модалка
   покажет только базовый расчёт (факт / брак / нестандарт / к оплате).»

Показать минимум две карточки:
- Огурцы — три заполненных параметра:
  «Калибр 6–9 см» (кг, детализирует вес к оплате)
  «Калибр 9–12 см» (кг, детализирует вес к оплате)
  «Калибр 12+ см» (кг, дублирует поле Нестандарт)
- Томаты — блок пустой, плейсхолдер «Параметры качества не заданы».

Никакие другие части справочника не трогать.
