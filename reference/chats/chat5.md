Система учёта отгрузок овощного сырья от поставщиков на переработку.
Текущее состояние — в файле Wireframes.html в проекте.

ВИЗУАЛЬНЫЙ ЯЗЫК (сохранять во всех новых экранах):
- Палитра сырья: огурцы — зелёный, черри — красно-розовый,
  томаты — оранжевый, патиссоны — жёлтый, халапеньо — тёмно-зелёный,
  перец — красно-зелёный градиент, баклажан — фиолетовый.
- Чипы статуса: «Запланировано» — жёлтый + полосатая штриховка строки,
  «Отправлено» — зелёный.
- Пилюли сырья, оверлейный скроллбар, типографика — как в существующих
  экранах.
- Tailwind через CDN, иконки lucide или эмодзи.

РОЛИ:
- Пользователь: главная (просмотр, поиск, фильтры, колонки,
  toggle Таблица/Heatmap, Excel, печать), справочник «Водители»
  (просмотр + поиск + фильтр по ТК), модалка карточки водителя,
  мобильный просмотр.
- Админ: всё вышеперечисленное + создание/редактирование везде,
  тара, ингредиенты, контракты, аналитика, остальные справочники.

ПРАВИЛА РАБОТЫ:
- Не переписывай существующие экраны, если в задаче явно не сказано
  обратное.
- Новые экраны добавляй в конец файла, в логических разделах.
- Никакой реальной логики — статичные wireframes с моковыми данными.

ЗАДАЧА: создать НОВЫЙ раздел «Аналитика». Существующие экраны
не трогать.

ШАПКА ФИЛЬТРОВ
- Период: Сезон / Месяц / Неделя / Произвольный диапазон /
  Сравнение двух периодов (toggle).
- Поставщик, Сырьё, ТК.
- Кнопка «↓ Excel» (экспорт текущего среза).

KPI-ПЛИТКИ СВЕРХУ
- Отдельная плитка по каждому виду сырья + общая:
  Всего привезено за сезон (кг) · % плана ·
  Σ ₽ привезено / план ₽ · Активных контрактов.

ВИДЖЕТЫ
1. «Выполнение контрактов» — горизонтальные столбцы:
   контур = план, заливка цветом сырья = факт.
   Подпись «Генералов · Огурцы · 148 000 / 400 000 кг (37%)».
   Сортировка: по % выполнения / по объёму.
2. Общий % выполнения сезонного плана по каждому виду сырья
   (donut или радиальные индикаторы).
3. График поставок по дням/неделям (столбцы или линии),
   разрез по видам сырья.
4. Топ-поставщики — два рейтинга: по объёму и по % выполнения.
5. Heatmap «поставщик × неделя» (цвет — объём).
6. Рейсы по транспортным компаниям:
   - Прямые рейсы (с сырьём).
   - «Обратный рейс» (доставка тары/ингредиентов поставщику)
     — отдельная категория.
7. Таблица «Осталось машин до выполнения плана» по
   (поставщик × сырьё):
   План кг · Привезено кг · Остаток кг ·
   Средний вес рейса (авто + опц. ручное, показать оба) ·
   Машин осталось.
8. Лента алертов:
   - Контракт выполнен на 90%.
   - Нет поставок более 7 дней при активном контракте.
   - Перевыполнение на 110%.
9. Срез-конструктор «поставщик × сырьё × период» с экспортом.

Доступ: только админ.
